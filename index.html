Which Tile Should I Discard?
Supporting Beginners in Mahjong by Presenting Recommended Discards in Mixed Reality

Jun Kojima, Shizuka Shirai, Naoya Chiba, Yuki Uranishi
The University of Osaka

Video BibTeX

Abstract

This study proposes a novice Mahjong players support system using Mixed Reality to reduce the barriers that beginners face during physical play. Our system calculates an optimal discard ranking based on the playerâ€™s current hand and displays the recommended tile to discard via a hologram using an optical head-mounted display. To evaluate the effectiveness of the proposed system, we conducted a comparative experiment with novice players using a single-player Mahjong environment, with and without the support system.The results showed improved tile efficiency and high user satisfaction when using the system.
